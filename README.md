# Transcriptional dynamics in 4sU-labeled mESCs

This repository contains workflows developed during an MSc research internship to analyze 4sU-labeled scNT-seq data from mouse embryonic stem cells generated by Qiu et al. (2020).

The project includes quality control, assessment of potential 4sU-related quantification bias, analysis of transcriptional variability, and inference of transcriptional kinetic parameters.

## Repository structure

### `quality_control_and_4su_bias/`

Quality-control workflow for paired new- and old-RNA count matrices, including:

- cell- and gene-level quality control
- annotation of pluripotent, intermediate, and 2C-like cell states
- assessment of RNA-stability estimates
- checks for potential 4sU-related dropout and quantification bias
- export of processed data for downstream analyses

The 4sU-related quantification-bias analysis was adapted from the **grandR** workflow.

Cell-state annotation was performed only within this quality-control workflow.

### `fano_factor_analyses/`

Fano factor-based analyses of gene-level transcriptional variability in the pluripotent cell population.

The workflow examines mean-variance relationships and identifies genes with higher or lower variability than expected from their mean expression.

### `kinetic_inference_workflows/`

Adapted workflows for inferring transcriptional kinetic parameters in the pluripotent cell population.

- `nascseq2_adapted/`  
  Adapted from the NASC-seq2 analysis code described by Ramsköld et al. (2024).

- `deeptx_adapted/`  
  Adapted from the DeepTX framework described by Huang et al. (2026).

The project-specific adaptations include dataset preparation, parameter selection, filtering, workflow restructuring, visualization, and comparison of inferred kinetic parameters.

## Analysis scope

The quality-control workflow includes pluripotent, intermediate, and 2C-like cells.

The Fano factor and kinetic-inference analyses were restricted to cells classified as pluripotent.
