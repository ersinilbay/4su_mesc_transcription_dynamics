# Master-internship


This repository contains workflows developed from my MSc internship work on 4sU-labelled single-cell transcriptomics data in mouse embryonic stem cells (mESCs).

- `refactored_qc_workflow/`  
  A refactored QC workflow for paired new and old RNA count matrices, including quality control, cell-state annotation, RNA-stability validation, and export of processed outputs.

- `refactored_fano_workflow/`  
  A refactored workflow for Fano factor-based analyses, used to compare transcriptional variability across genes, conditions, and cell states.

- `NASC-Seq2 code` and `deeptx-start`  
  Supporting scripts related to bursting parameter estimation and comparison with external datasets in wild-type mESCs cultured under serum and 2i conditions.
