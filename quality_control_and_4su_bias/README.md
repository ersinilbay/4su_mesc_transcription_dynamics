# Quality control and 4sU bias assessment

This directory contains the quality-control workflow for 4sU-labeled scNT-seq data from mouse embryonic stem cells generated by Qiu et al. (2020).

The workflow includes:

- construction of layered `AnnData` objects from new- and old-RNA count matrices
- cell- and gene-level quality control and filtering
- normalization, clustering, and marker-based annotation
- annotation of pluripotent, intermediate, and 2C-like cell states
- assessment of RNA-stability estimates
- assessment of potential 4sU-related dropout and quantification bias, adapted from grandR
- export of processed data for downstream analyses

## Files

- `run_qc_report.py`: runs the complete workflow
- `config.py`: defines paths and analysis settings
- `io_utils.py`: loads data and saves outputs
- `pipeline.py`: contains the analysis steps
- `plotting.py`: contains plotting functions

Input data are not included in this repository.
